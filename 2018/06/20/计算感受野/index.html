计算感受野

发表于 2018-06-20 | 分类于 caffe
  • feature map的计算公式为outsize = (insize - fsize + 2*pad) / stride + 1
  • stride 的计算公式为totstride = totstride * stride，这两个公式都是从前往后算
  • receptive field RF = ((RF -1)* stride) + fsize，这个要从后往前算
